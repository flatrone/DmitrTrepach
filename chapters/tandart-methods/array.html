Массивные методы

Добавление элементов

arr.push(элемент, элемент, ...) - добавляет элемент в конец списка
arr.unshift(элемент, элемент, ...) - добавляет элемент в начало списка

Удаление элементов

arr.shift() - удаляет первый элемент из массива
arr.pop() - удаляет последный элемент из массива

Нарезка

arr.slice(откуда, [докуда]) - отрезае массив

splice

arr.splice(откуда, сколько, [вставить элемент], [вставить элемент]...) - удаляет и взамен вставляет элементы

indexOf

arr.indexOf("элемент") - ищет индекс элемента

\

Наличие элемента в массиве

arr.includes(элемент) - вернет true или false
