Строковые методы

Регистры

string.tUpperCase()
string.toLowerCase()

substr, substring, slice

string.substr(откуда, [сколько]) - отрезает строку
str.substring(откуда, [докуда]) - отрезает строку
str.slice(откуда, [докуда]) - отрезает строку

indexOf

str.indexOf(что искать, [откуда искать]) - возвращает индекс искомого
str.lastIndexOf(что искать, [откуда искать]) - возвращает последний индекс

Начало и конец строки

str.startsWidth(что искать, [откуда]) - начинается ли строка с ...
str.endsWidth(что искать, [откуда]) - заканчивается ли строка с ...

Разделение символов

str.split(разделитель, [максмальное кол-во элементов]) - разбивает строку по разделителю

Объединенме симовлов

arr.join(объеденитель) - объединяет массив в строку
