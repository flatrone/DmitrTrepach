Массивы и объекты с анонимными функциями

Так как фунцкции ведут себя как строки или числа, можно сделать массив из функций 

    let arr = [
	function() {alert('1')},
	function() {alert('2')},
	function() {alert('3')},
];
 

чтобы вызвать РАБОТАЮЩУЮ функцию, то: 


alert(arr[0]()) // внимание на скобки


Воспользуемся нашим объектом:


let obj = {
	func1: function() {alert(1)},
	func2: function() {alert(2)},
	func3: function() {alert(3)},
};

obj.func1(); // выведет 1
