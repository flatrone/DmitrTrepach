Пусть есть функция, которая возврощает другую функцию

function test() {
	return function() {
		
	}
}


Если родительская функция имеет какие-то переменные, то эти переменные будут 
содержаться в лексическом окружении(доступных переменных) возврощаемой функции:

function test() {
	let num = 1; // переменная родительской функции
	
	return function() {
		// лексическое окружение = {num: 1}
	}
}
